# Music & Motivation - Data Analysis! (Python)

This repo analyzes data collected by the Pre-Partiers for a project in COGS 164 (Neurobiology of Motivation) about music and social behaviors at parties. The repo analyzes the following:
- survey data about music preferences, party environments, and motivation/social behavior
- musical properties of song choices from survey
- musical properties of songs recorded at parties (naturalistic observations)

## Folder structure
- `data/raw/` — original exports (**gitignored**; may contain identifying info)
- `data/processed/` — cleaned + anonymized CSV used for analysis
- `notebooks/` — Jupyter notebooks with analysis and steps
- `src/` — reusable scripts
- `reports/` — short markdown summaries of data
- `figures/` — saved plots
